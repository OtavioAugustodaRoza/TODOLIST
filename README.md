## sobre o projeto

Essa lista de tarefas foi atividade de programação do curso técnico que estou fazendo dai pra deixar o rep arrumado to fazendo esse readme enfim espero que goste.

é uma todolist feita em **Vue.js** — sem frescura, sem dependência desnecessária, só o essencial pra você organizar o que precisa ser feito. adiciona tarefa, marca como concluída, remove. vida simples.

o foco foi em manter a interface limpa e o código legível.

-----

## funcionalidades

-    adicionar novas tarefas
-    marcar tarefas como concluídas
-    remover tarefas da lista
-    filtrar entre todas, ativas e concluídas
-    persistência via `localStorage` (seus dados não somem ao recarregar)

## como rodar

pré-requisitos: **Node.js 18+** e **npm** ou **yarn** instalados.

```bash
# clone o repositório
git clone https://github.com/seu-usuario/todolist-vue.git

# entre na pasta
cd todolist-vue

# instale as dependências
npm install

# rode em desenvolvimento
npm run dev
```

a aplicação vai estar disponível em `http://localhost:5173`
